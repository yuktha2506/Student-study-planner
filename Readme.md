# StudyBloom -- Study Planner Web App

## Overview

StudyBloom is a web-based study planner that helps students organize
tasks, subjects, study sessions, and reminders in one place. The
application runs in the browser and stores data using LocalStorage.

## Features

-   Dashboard with study statistics
-   Task manager with priorities
-   Weekly timetable planner
-   Progress tracking
-   Reminder notifications
-   Subject management with colors

## Project Structure

StudyBloom/ - index.html - style.css - script.js - README.md

## Technologies

HTML5, CSS3, JavaScript (Vanilla JS), LocalStorage

## How to Run

1.  Download the project
2.  Open index.html in a browser
